# PCleaner

专业存储清理工具，基于 C# WPF 开发。

## 功能特点
- **深层扫描**：识别系统临时文件、应用缓存、开发工具缓存（npm, NuGet, Pip, Maven, Gradle）以及浏览器缓存。
- **三级分类**：
    - 🟢 **绿色**：安全项，可一键清理。
    - 🟡 **黄色**：需人工判断项（如下载文件夹）。
    - 🔴 **红色**：风险项（仅展示，暂不支持直接删除）。
- **可选回收站**：支持将文件移至回收站或彻底删除。
- **现代化 UI**：简洁直观的 WPF 界面，支持磁盘占用实时概览。

## 快速启动
双击 `start.bat` 即可自动编译并运行程序。

## 技术栈
- .NET 10.0 (WPF)
- CommunityToolkit.Mvvm
- Windows Shell API (SHFileOperation)
